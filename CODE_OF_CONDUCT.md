# Code of Conduct

This project follows the
[Contributor Covenant Code of Conduct, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

To report a violation, open a private report via
[GitHub's report abuse form](https://github.com/contact/report-abuse) or
contact the maintainers listed in [MAINTAINERS.md](MAINTAINERS.md).
